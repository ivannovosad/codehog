# Privacy Policy

Code Hog is a Claude Code plugin that runs entirely within your local Claude Code session.

## Data Collection

Code Hog does **not**:
- Collect, transmit, or store any user data
- Make external network requests
- Send code or analysis results to any third-party service
- Use analytics, telemetry, or tracking of any kind

## How It Works

Code Hog provides a skill prompt that instructs Claude to analyze your source files locally. All processing happens within your existing Claude Code session. No data leaves the environment Claude Code already operates in.

## Contact

For questions about this policy, open an issue at https://github.com/ivannovosad/codehog/issues
